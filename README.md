## üìñ About

This project is a unique take on a personal portfolio, presenting information through a simulated terminal interface. It features a retro aesthetic combined with modern interactive elements like a physics-based 3D ID card. Users can type commands to navigate or click through the interface to discover details about my skills, projects, and background.

## ‚ú® Features

- **Interactive Terminal**: Type commands like `help`, `projects`, and `about` to interact.
- **Physics-Based 3D Card**: A draggable ID card with realistic lanyard physics and 3D tilt effects.
- **Responsive Design**: Optimized for both desktop and mobile viewing with a flexible layout.
- **Retro Aesthetic**: Custom CRT-style styling, blinking cursor, and monospaced typography.
- **Command History**: Navigate through previous commands just like a real terminal.

## üöÄ Run Locally

**Prerequisites:** Node.js (v18+ recommended)

1. **Clone the repository:**
   ```bash
   git clone https://github.com/goyalpuneet18/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
   Navigate to `http://localhost:3000/`.
